Used DESeq2 vignette to write analysis script: https://www.bioconductor.org/packages/devel/bioc/vignettes/DESeq2/inst/doc/DESeq2.html Note: Log fold change shrinkage for visualization and ranking: apeglm is the adaptive t prior shrinkage estimator from the apeglm package (Zhu, Ibrahim, and Love 2018). As of version 1.28.0, it is the default estimator.

  mutate(condition = ifelse(condition == "log2FoldChange.x", "LPS", "VNS"),

  mutate(padj = paste0(logp.x,"_", logp.y),

                       str_split_i(padj, pattern = "_", i = 1),
                       str_split_i(padj, pattern = "_", i = 2)),
         pLabel = ifelse(padj > 1.3, "*", ""),
         labelC = ifelse(log2FC > 0, "black", "white"))

  geom_tile(color = "white") +theme_classic()+
